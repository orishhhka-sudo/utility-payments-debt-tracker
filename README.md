Разработка класса для учета и анализа задолженности по квартплате в ЖЭС (Жилищно-эксплуатационная служба).
Программа моделирует работу ЖЭС и рассчитывает задолженность жильцов. Также программа распределяет этот долг по трем временным интервалам просрочки.
Данная программа решает задачу автоматизации учета и анализа задолженности по коммунальным платежам для отдельно взятого ЖЭС (жилищно-эксплуатационной службы).
